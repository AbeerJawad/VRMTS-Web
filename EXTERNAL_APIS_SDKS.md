# External APIs/SDKs

| API/SDK and Version | Description | Purpose of Usage | API Endpoint/Function/Class |
|---|---|---|---|
| **Three.js** (v0.180.0) | 3D graphics library | Real-time 3D visualization, rendering, interactions | `THREE.Scene()`, `THREE.WebGLRenderer()`, `THREE.Raycaster()`, `THREE.Box3()` |
| **Three.js FBXLoader** | FBX model loader | Loading 3D anatomical models | `FBXLoader()`, `loader.load(filePath)` |
| **Axios** (v1.13.1) | HTTP client library | API requests to backend for quizzes and data | `axios.get()`, `axios.post()`, `axios.put()`, `axios.delete()` |
| **React Query** (v5.90.5) | Server state management | Caching and syncing server state | `useQuery()`, `useMutation()` |
| **React Router DOM** (v7.9.4) | Client-side routing | c | `<BrowserRouter>`, `<Routes>`, `useNavigate()` |
| **Express.js** (v5.1.0) | Web framework for Node.js | Building RESTful API backend | `app.get()`, `app.post()`, `app.put()`, `app.delete()` |
| **MySQL2** (v3.15.3) | MySQL database driver | Database connectivity and queries | `mysql.createConnection()`, `connection.query()` |
| **Express Session** (v1.18.2) | Session management middleware | Managing user sessions | `session()`, `req.session` |
| **Bcryptjs** (v3.0.2) | Password hashing library | Secure password storage and verification | `bcryptjs.hash()`, `bcryptjs.compare()` |
| **Helmet.js** (v8.1.0) | Security middleware | HTTP security headers protection | `helmet()` |
| **CORS** (v2.8.5) | Cross-origin middleware | Enable frontend-backend communication | `cors()` |
| **TailwindCSS** (v4.1.16) | CSS utility framework | Responsive UI styling | CSS utility classes |
| **Radix UI** (v2.1.7) | UI primitives | Accessible form components | `<Label>`, `<Slot>` |
| **Dotenv** (v17.2.3) | Environment loader | Load configuration from `.env` file | `dotenv.config()` |
